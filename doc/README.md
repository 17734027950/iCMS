欢迎使用 iCMS 内容管理系统
==========

> iCMS 是一套采用 PHP 和 MySQL 构建的高效简洁的内容管理系统,为您的网站提供一个完美的开源解决方案

## iCMS 可以用来干什么
- 博客
- 资讯/新闻站点
- 公司产品展示
- 图片展示
- 快速二次开发

本说明将全面的介绍iCMS的模板系统。
